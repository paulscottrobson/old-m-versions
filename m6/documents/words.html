! (macro)

Write b to memory[a] 16 bit

* (macro)

Multiply a by b, result in a, b unchanged.All values are 16 bit so overflow is lost

+ (macro)

Add b to a, b unchanged

+! (word)

Add b to memory[a] 8 bit

- (macro)

Subtract a from b, result in a, b unchanged.Note ; this is a 7 byte macro ; use 1- 2- or adding negative constants where possible

-! (word)

Sub b from memory[a] 8 bit

/ (word)

Divide a into b, placing the result in a.B is unchanged.Division by zero does not cause an error but the result is not predictable

0- (word)

2's complement, e.G.Negate a

0< (word)

True if a < 0,false otherwise

0= (word)

True if a = 0,false otherwise

1+ (macro)

Add 1 to a

1- (macro)

Sub 1 from a

2* (macro)

Double a

2+ (macro)

Add 2 to a

2- (macro)

Sub 2 from hl

2/ (macro)

Halve a arithmetically

; (macro)

Return from call

< (word)

Set a to b < a ; e.G.A-b > 0, values 0 (false) or -1 (true)

<= (word)

Set a to b <= a ; e.G.A-b >= 0, values 0 (false) or -1 (true)

<> (word)

Set a to (a != b), values 0 (false) or -1 (true)

= (word)

Set a to (a == b)

> (word)

Set a to b > a ; e.G.A-b < 0, values 0 (false) or -1 (true)

>= (word)

Set a to b >= a ; e.G.A-b <= 0, values 0 (false) or -1 (true)

@ (macro)

16 bit read of a into a (unary operators)

a>b (macro)

Copy a to b

a>r (macro)

Push a on return stack

abs (word)

Absolute value of a

and (word)

Bitwise and b into a, b unchanged

b>a (macro)

Copy b to a

breakpoint (macro)

Compiles a cspect specific breakpoint here

bswap (macro)

Swap bytes of a

buffer (slow)

Put the buffer address in a and size in b

c! (macro)

Write b to memory[a] 8 bit

c@ (macro)

8 bit read of a into a (unary operator)

clrscreen (slow)

Clears the display

copy (word)

Copy memory [a+0]:source [a+2]:target [a+4]:count

cursor! (slow)

Position cursor at screen location a (000-2ff)

debug (slow)

Display a+b at the bottom right

false (macro)

Put false (0) in a

fill (word)

Fill memory [a+0] byte [a+2] target [a+4] count

halt (macro)

Stop processor

inkey (slow)

Scans the keyboard looking for any key pressed.Return in a 0 if no key pressed

max (word)

Put in a the larger of a and b

min (word)

Put in a the smaller of a and b

mod (word)

Divide a into b and put modulus into a, b is unchanged

not (word)

1's complement a

or (word)

Bitwise or b into a, b unchanged

pop.ab (macro)

Pop ab off stack

pop.bb (macro)

Undo pushab preserving a (for return values)

push.ab (macro)

Push a and b on the stack

r>a (macro)

Pop return stack to a

screen! (slow)

Write b to screen location a (000-2ff).B is the 2 + 6 bit format, the lower 6 bits being the 6 bit ascii value and the upper two representing the colour (0=red,1=white,2=green, 3=yellow)

swap (macro)

Swap a and b

system.info (slow)

Get address of data area

system.xmacro (slow)

Expand macro.The top of the return stack points to the count followed by the macro data, the one after is the actual return

true (macro)

Put true (-1) in a

xor (word)

Bitwise xor b into a.B is unchanged
